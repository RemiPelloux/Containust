Guide

From Docker Compose

Same mental verbs — plan, build, run — without a privileged daemon. Convert with ctst convert, then tighten image pins.

What changes

Aspect Docker Compose Containust
Daemon dockerd as root No daemon — syscalls + state file
Images Online-first Hub pulls Local-first; remote opt-in + digests
Rootfs Writable by default Read-only by default
Dependencies depends_on + manual env CONNECT with injected host/port
Air-gap Manual First-class --offline

Convert in one command

ctst convert docker-compose.yml > stack.ctst
ctst plan stack.ctst
# Pin images (pull + replace tags with image://…@sha256:…)
ctst pull nginx:alpine
ctst build stack.ctst
ctst run stack.ctst --detach
Review the draft. The converter maps structure; you still choose local vs registry sources and tighten secrets / volumes for production.

Syntax cheat sheet

Compose .ctst
services: COMPONENT name { }
image: nginx:1.25 image = "tar:///opt/images/nginx.tar" or digest pin
ports: ["8080:80"] port = 80 + EXPOSE 8080:80
volumes: ["./data:/data"] volume = "./data:/data" or volumes = […]
environment: env = { K = "V" }
depends_on: [db] CONNECT app -> db
restart: unless-stopped restart = "always"
healthcheck: healthcheck = { command = […] }
mem_limit: 512m memory = "512MiB"
read_only: true readonly = true (already the default)
docker compose up ctst run stack.ctst
docker compose ps ctst ps
docker exec -it app sh ctst exec app -- /bin/sh

Before / after

Compose

services:
  web:
    image: nginx:alpine
    ports:
      - "8080:80"
    depends_on:
      - api
  api:
    image: ghcr.io/acme/api:1.2
    environment:
      DATABASE_URL: postgres://db:5432/app
    depends_on:
      - db
  db:
    image: postgres:16
    environment:
      POSTGRES_PASSWORD: secret

.ctst

COMPONENT db {
    image = "image://library/postgres@sha256:…"
    port  = 5432
    env   = { POSTGRES_PASSWORD = "secret" }
}

COMPONENT api {
    image = "image://ghcr.io/acme/api@sha256:…"
    env   = { DATABASE_URL = "postgres://${db.host}:${db.port}/app" }
}

COMPONENT web {
    image = "image://library/nginx@sha256:…"
    port  = 80
    ports = ["8080"]
}

CONNECT api -> db
CONNECT web -> api
EXPOSE 8080:80
